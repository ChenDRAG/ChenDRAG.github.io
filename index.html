Projects

A Video-based Virtual Try-on System for Non-rigid Clothing

  • Propose a video-based virtual try-on system for non-rigid(half-rigid) clothing.
  • Leverage both reconstructed 3d human model representations and a learned deep renfinement network to synthesis (near) video-realistic try-on results.
  • First attempt to try adding dynamic details to clothes in virtual try-on systems using a generative adversarial network.

A Cheating device for Wechat game “Tiaoyitiao”

This cheating device use one camera and one Raspberry Pi to take picture of your smartphone. Photos are then analysed to control a servo so that it can press the screen for the right time. I did this project just for fun when I was in summer vacation, 2018. One interesting fact is that I found out sausages are the most desirable meterial for pressing the screen: They are conductive, cheap and most importantly, delicious.

A tangram solver based on heuristic algorithm

We implement heuristic algorithm to design this tangram solver, which can solve any 13-pieces puzzle in less than one second using a raw picture as input. A technical speech of this project is delivered in front of the class for Artificial Intelligence.
